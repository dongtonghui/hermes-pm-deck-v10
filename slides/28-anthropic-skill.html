Skill 规范

一个 Skill 是一个包含 SKILL.md 的目录。结构清晰、可移植、可被任何兼容工具调用。

目录结构

skill-name/
├── SKILL.md      # 核心说明文档
├── README.md     # 可选：使用说明
└── ...           # 可选：脚本、模板、示例

SKILL.md 标准结构

Frontmatter：name + description，说明 Skill 的调用名称和触发场景。

01 · 何时使用

描述使用场景和触发条件，让 AI 知道什么时候该调用这个 Skill。

02 · 输入

明确需要用户提供什么信息，避免 AI 自己编造。

03 · 工作流

分步骤说明 Skill 的执行流程，让输出可预期、可调试。

04 · 输出格式 + 示例 + 注意事项

明确输出结构、给一个输入/输出样例、列出边界条件和常见错误。
