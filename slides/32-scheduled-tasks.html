Scheduled Tasks

定时任务：让 AI 在后台持续运转

把重复工作交给 AI，你只需要设定规则和审阅结果。

01

触发条件

什么时候运行？

例：每天 9:00 / 每周一 / 每月 1 号 / 事件发生
02

输入来源

从哪里获取数据？

例：网页 / API / 本地文件 / 数据库 / 邮件
03

输出格式

生成什么形式的结果？

例：Markdown / JSON / Excel / 图表 / 飞书消息
04

推送渠道

结果送到哪里？

例：邮件 / 飞书 / 钉钉 / Slack / 微信

一个定时任务 = 触发条件 + 输入来源 + 输出格式 + 推送渠道
