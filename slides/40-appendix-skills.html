附录 A · 本次工作坊会用到的 Hermes 技能

下面两个技能都已在主讲人的 Hermes 中安装。它们代表了「把 AI 从聊天框变成工作流」的两条路径：一个负责快速产出高保真视觉，一个负责把社区声音变成可分析的数据。

H

huashu-design

花叔 Design：用 HTML 做高保真原型、交互 Demo、幻灯片和动画。
工作流程原理
  • 1 先澄清目标：受众、尺寸、情感基调、输出格式
  • 2 收集真实资产：品牌色、logo、产品图、字体
  • 3 多方向探索：并行产出 3 套差异化视觉方案
  • 4 用 HTML/CSS 交付可点击、可演示、可导出 MP4/GIF 的成品
适用场景
产品 mockup、PPT 幻灯片、交互动画、设计变体评审。
下载与安装
安装：将仓库 clone 到 ~/.hermes/skills/creative/huashu-design/
R

reddit-community-research

系统性爬取 Reddit 社区数据，提取痛点、情绪信号与功能需求优先级。
工作流程原理
  • 1 确认目标：subreddit、关键词、时间范围、数据量
  • 2 用 pullpush.io 或 Tavily 获取帖子与评论数据
  • 3 结构化：过滤噪音、分类痛点、提取情绪关键词
  • 4 输出报告：痛点矩阵、需求优先级、可直接引用的高保真原声
适用场景
竞品社区调研、用户需求验证、社交媒体聆听、Reddit 情报。
下载与安装
安装：将仓库 clone 到 ~/.hermes/skills/research/reddit-community-research/
如何安装 Hermes 技能： 将 skill 仓库 clone 到 ~/.hermes/skills/<category>/<skill-name>/ 目录下，或直接通过 Hermes 内置 skill 管理命令安装。安装后重启 Hermes 或运行 hermes skills list 即可看到新技能。
