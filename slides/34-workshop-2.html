实战任务二：封装一个自己的 Skill

把你每周重复做的一次工作，按 Anthropic 规范写出 SKILL.md 框架。

选题方向

选择一个你每周至少做两次的重复性工作，例如：

  • 竞品动态监控
  • 用户反馈分类
  • PRD 初稿生成
  • 周报数据整理
  • 会议纪要的 action items 提取

任务

  • 写出 Skill 名称和 description
  • 定义输入字段
  • 列出 3-5 个工作流步骤
  • 描述输出格式
  • 标注一个注意事项
