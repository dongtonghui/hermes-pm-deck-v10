Hermes Mechanism

Hermes 的 Skill 机制与 Anthropic Skill 规范异同

Hermes 采用了类似 Anthropic Claude Code Skills（SKILL.md）的描述机制，并在此基础上扩展了定时任务、消息渠道、MCP 工具等能力。它们不是同一套实现，但理念相通。

01

类似理念

都用 SKILL.md 描述 name、description、工作流，让自然语言指令可复用。

02

Hermes 扩展

支持定时触发、消息渠道推送、浏览器自动化、跨会话记忆。

03

可迁移

写好的 Skill 可以在不同项目、不同团队成员之间复用。
